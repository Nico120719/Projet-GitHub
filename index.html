Visual Studio Code

  • Accessibilité
    • Édition de base (Community Edition) disponible sans frais (open source)
    • Édition complète offrant toutes les caractéristiques du logiciel ceoendant un peu coûteuse
    • Depuis 2019, disponible en 14 langues officielles
  • Richesse Fonctionnelle
    • Offre le service Live Share qui permet aux développeurs de travailler à distance sur un même projet
    • Intégration facile avec Git
  • Capacités de l'Éditeur
    • Puissant éditeur muni des fonctionnalités IntelliSense et IntelliCode, avec complétion intuitive de code
    • Modèles (templates) disponibles, utiles pour les nouveaux programmeurs
    • L'extension Live Server s'avère pratique pour modifier le code et voir le résultat en temps réel
  • Possibilité d'ajouter des Plug-ins
    • Multitude de plug-ins disponibles, disponibles sur le Visual Studio Marketplace
  • Langages de Programmation supportés
    • 36 langages de programmation, incluant C#, C++, F#, JavaScript, Python, TypeScript, PHP, Visual Basic, XAML
    • Requiert une configuration supplémentaire pour le langage Java
  • Capacités du Débogueur
    • Peut être déclenché lorsqu'une application en marche à l'extérieur de l'environnement encourt des problèmes
    • Permet de créer des points de rupture (breakpoints) utiles pour rechercher une variable ou condition problématique avec le code
  • Présence de Frameworks de Tests
    • Pour .NET, Intellitest peut générer des tests unitaires. Une analyse de cas est effectuée pour chaque branche du code
    • Accès à une robuste plateforme de tests grâce aux multiples plug-ins
  • Développement Web
    • Élu meilleur IDE pour JavaScript par TMS Outsource et voté le plus populaire par State Of JS dans un sondage de 2018
    • Efficace pour les applications Web et Cloud. Se connecte aisément aux bases de données
